SEQUENCE DIAGRAM SEDERHANA PER FITUR SIMASJID
=============================================

1. SEQUENCE DIAGRAM LAPORAN KAS
-------------------------------
```plantuml
@startuml
skinparam defaultFontSize 16
skinparam backgroundColor white
skinparam defaultFontName Arial
skinparam ParticipantBackgroundColor white
skinparam ParticipantBorderColor black

participant "Admin" as Admin
participant "Halaman Kas" as HalamanKas
participant "Form Data" as FormData
participant "Database" as Database

Admin -> HalamanKas: Akses Menu Kas
activate HalamanKas
HalamanKas -> Database: Load Data Kas
activate Database
Database --> HalamanKas: Return Data
deactivate Database
HalamanKas --> Admin: Tampilkan Daftar Kas
deactivate HalamanKas

Admin -> FormData: Pilih Aksi CRUD
activate FormData
Admin -> FormData: Input/Edit Data
FormData -> Database: Simpan Data
activate Database
Database --> FormData: Konfirmasi
deactivate Database
FormData --> Admin: Berhasil Disimpan
deactivate FormData

@enduml
```
