SEQUENCE DIAGRAM PER FITUR SIMASJID
===================================

1. SEQUENCE DIAGRAM LAPORAN KAS
-------------------------------
```plantuml
@startuml
skinparam monochrome true
skinparam defaultFontName Arial
skinparam ParticipantBackgroundColor white
skinparam ParticipantBorderColor black

actor Admin
participant "Web" as Web
participant "KasController" as Controller
participant "LaporanKas" as Model

Admin -> Web: akses menu kas
Web -> Controller: getKasList()
Controller -> Model: ambil data kas
Model --> Controller: data kas
Controller --> Web: tampilkan data
Admin -> Web: tambah/edit/hapus transaksi
Web -> Controller: simpan perubahan
Controller -> Model: update data
Model --> Controller: hasil
Controller --> Web: refresh data
@enduml
```

5. SEQUENCE DIAGRAM JANJI TEMU USTADZ
-------------------------------------
```plantuml
@startuml
skinparam monochrome true
skinparam defaultFontName Arial
skinparam ParticipantBackgroundColor white
skinparam ParticipantBorderColor black

actor User
actor Admin
participant "Web" as Web
participant "JanjiTemuController" as Controller
participant "JanjiTemu" as Model

User -> Web: akses menu janji temu
User -> Web: input data & submit
Web -> Controller: simpan permintaan
Controller -> Model: simpan data
Model --> Controller: hasil
Controller --> Web: notifikasi admin
Admin -> Web: review permintaan
Admin -> Controller: setujui/tolak
Controller -> Model: update status
Model --> Controller: hasil
Controller --> Web: notifikasi user
@enduml
```
